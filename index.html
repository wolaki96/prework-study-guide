HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy to navigate the webpage.
  • Elements are used to divide the content on the page and to describe the content to browsers.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS defines the color, aesthetics, visual layout, and organization of the webpage.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS is used for styling an element directly in the HTML file.
  • The internal style sheet contains rules for the webpage in the head section of the HTML file.
  • The external style sheet allows developers to keep all their CSS rules in separate files. This is the preffered CSS method.

Git

  • Git is a free and open source software for tracking changes in a set of files.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code .: opens the vs script
  • cd: stands for "change directory: It takes us to the file we are searching for
  • pwd: prints the present working directory
  • git add -A: commits all files in the working branch

JavaScript

  • JavaScript is a high level dynamic programming language that allows users to interact with websites.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A semicolon at the end of the line indicates where a statement ends.
  • You can name a variable anything, just make sure you check for any restrictions.
  • JavaScript is case sensitive. When debugging, check for something as simple as case first!
  • There are different types of variables. String (sequence of text), Boolean (true/false), and Array (multiple value storage in a single reference).
  • You can use comments that the browser ignores to help you write notes and keep track of changes.
